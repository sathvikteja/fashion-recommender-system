# 👕 Fashion Recommender System

A content-based image recommendation system using deep learning.

## 🚀 Features
- Upload an image
- Get visually similar fashion items
- Built using ResNet50 embeddings

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- Scikit-learn

## ⚙️ How it works
1. Extract features using ResNet50
2. Store embeddings
3. Use Nearest Neighbors for similarity search

## ▶️ Run locally
```bash
pip install -r requirements.txt
streamlit run main.py
